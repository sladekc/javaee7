## RESTful Services



### Annotationen
* HTTP
    * @GET
    * @POST
    * @PUT
    * @DELETE
    * @HEAD
    * @OPTIONS
* Pfad
    * @Path

* Parameter

    * @QueryParam
    * @CookieParam
    * @FormParam
    * @HeaderParam
    * @MatrixParam
* MIME
    * @Consumes
    * @Produces
* Sonstige 
    * @Provider

<http://localhost:8080/rest/users/Hans%20Dampf>
